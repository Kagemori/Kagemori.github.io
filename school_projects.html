Master Thesis: Optimization of Facial Detection

Master Thesis on object detection, how as a programmer entering such a field can begin the machine learning pipeline of collecting and cleaning data for training, the various models that can be used for training an object dection model and the pros and cons of each kind. I had used Kaggle to collect my dataset and if the model could not use the dataset, manually clean the set to be properly used for training. The main 3 models I focused on were Detectron2, Darknet and DETR, using the WIDERFACE, FDDB and CrowdHuman datasets.


Scikit Learn Clustering Project [GitHub For Lists]

Project making use of Scikit Learn's Clustering Algorithm. This project started with using a video game data set from Kaggle and creating a clustering algorithm to generate a "recommendation" list of games a user could enjoy playing based on what they have played previously and their personal rating of how much they liked the game.
